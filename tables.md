# 📋 Database Tables

The LearnFlow database is designed to manage users, personalized learning roadmaps, assessments, progress tracking, projects, certifications, and AI-powered learning interactions.

### Core Tables

| Table Name | Purpose |
|------------|----------|
| Users | Stores user account information |
| Skills | Stores available skills and domains |
| Learning_Roadmaps | Stores personalized learning paths |
| Phases | Stores roadmap phases and milestones |
| Concepts | Stores micro-learning concepts |
| Assessments | Stores quizzes and exams |
| Questions | Stores dynamic question bank |
| User_Attempts | Stores assessment results |
| Progress | Tracks user learning progress |
| Projects | Stores project requirements |
| Project_Submissions | Stores user project submissions |
| AI_Evaluations | Stores AI-generated project feedback |
| Certificates | Stores earned certificates |
| Achievements | Stores badges and rewards |
| User_Achievements | Maps achievements to users |
| AI_Mentor_Chats | Stores AI mentor conversations |
| Skill_Analytics | Stores mastery and performance metrics |

### Database Technologies

- **Supabase PostgreSQL** – Primary Database
- **Supabase Authentication** – User Authentication
- **Supabase Storage** – File & Certificate Storage

### Database Capabilities

- User Management
- Learning Roadmap Management
- Skill Tracking
- Dynamic Assessments
- Progress Monitoring
- Project Evaluation
- Certificate Generation
- AI Mentor Interactions
- Performance Analytics
